# Dental Clinic Management System
- A desktop-based clinic management system built using Java and MySQL.
- The system is designed to help manage clinic operations including patient records, appointments and administrative tasks.

## Technologies Used
- Java
- MySQL
- NetBeans IDE
- Maven

## Features
- Patient registration and management
- Appointment tracking
- Reports generation
- Clinic administrative functions
- Database integration for secure data storage

## Project Structure 
- 'src/' - Java source code
- 'lib/' - external libraries
- 'pom.xml' - Maven project configuration
